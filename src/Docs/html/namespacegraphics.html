def graphics.loadAnimation (   xmlPath,
  animation 
)

Loads an animation from a file and returns an Animation object.

Supports the following tags:

  • delay - Sets the default delay between frames for this animation, in seconds.
  • nextAnimation - Sets the animation which should occur after this animation, given no change in state.
  • Frame - Subobject which contains AnimationFrame data.

The Frame subobject supports:

  • image - Sets the image or sprite for this frame.
  • number - Sets the frame number. Auto numbers if not defined.
  • delay - Sets the delay between this frame and the next, in seconds.
Note

Frames will be linked in the order they are listed.

i.e. The first frame listed, will be displayed first. The second frame listed, will be displayed second. etc.

Parameters
xmlPathPath to xml file containing animation data.
animationName of animation to load from file.
